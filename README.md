# Socratic-Microteaching-Agent
本系统专为师范生微格教学设计，通过 DeepSeek-V3 API 作为决策大脑，结合本地 RAG (检索增强生成) 技术与多模态分析，实现对微课视频的深度解构与苏格拉底式教学反馈。 This system is designed for pre-service teachers' micro-teaching practice. It utilizes DeepSeek-V3 as the reasoning engine, combined with local RAG and multi-modal analysis to provide Socratic feedback on teaching videos.
